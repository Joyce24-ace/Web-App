How to use the SK Profiling Management System

This guide will help you navigate and utilize all features of the KK Profiling Management System effectively.

Getting Started
  1. Log in using your assigned username and password.
  2. You will be directed to the Dashboard showing the youth statistics.
  3. Use the sidebar navigation to access different sections.
  4. Click Logout in the top right corner when finished.

Dashboard

The Dashboard provides a quick overview of your barangay's youth data:

  • View total youth count and breakdown by status (Students, Working, Out of School)
  • See age distribution charts to understand youth demographics
  • Check gender distribution and key insights
  • Export reports by clicking the "Export Report" button
Youth Records Management

This is where you manage all youth profiles:

Adding a New Youth Profile:
  1. Click "Add New Youth" button
  2. Fill in all required fields marked with *
  3. Select the appropriate status (Student, Working, or Out of School)
  4. If status is "Working", provide occupation details
  5. Click "Save Profile" to add the record
Searching and Filtering:
  • Use the search bar to find youth by name or address
  • Use the filter dropdown to show specific status groups
Events Management

Create and manage SK events and activities:

  1. Click "Add Event" to create a new event
  2. Enter event title, date, time, location, and description
  3. Set the event status (Upcoming, Ongoing, or Completed)
  4. Save the event
SK Organization Chart

View the current SK officials and their contact information:

  1. Shows SK Chairperson at the top
  2. Displays Secretary and Treasurer
  3. Lists all SK Council Members with their assigned committees
  4. Contact numbers are provided for easy communication
Best Practices
  • Conduct youth profiling at least once a year
  • Update records immediately when youth status changes
  • Always verify contact information for accuracy
  • Export regular reports for SK meetings and planning sessions
  • Keep event information up to date for community awareness
  • Log out after each session to maintain security

Frequently Asked Questions

What is KK Profiling Management System?
The KK Profiling System is a digital platform designed to help Sangguniang Kabataan (SK) officials efficiently collect, manage, and analyze youth data in the barangay. It replaces the traditional manual process with a modern web and mobile application.
Who can use this system?
The system is designed for SK Chairpersons, SK Council Members, and designated System Administrators. Each user will have their own login credentials to access the system.
Can I delete a youth Profile?
Yes. In the Youth Records page, you can click the Edit icon to modify existing information or click the Delete icon to remove a profile. You can also view full details by clicking the Eye icon.
How do I search for specific youth?
Use the search bar in the Youth Records page to search by name or address. You can also filter records by status (Student, Working, or Out of School) using the dropdown filter.
What data is shown on the dashboard?
The Dashboard displays key statistics including total youth count, number of students, working youth, and out-of-school youth. It also shows charts for age distribution, status distribution, gender distribution, and key insights.
How do I create an event?
Navigate to the Events page, click "Add Event", fill in the event details (title, date, time, location, description, and status), then save. Events will be categorized as Upcoming or Past based on their status.
Can I export the data?
Yes. In the Dashboard, click the "Export Report" button to generate PDF or Excel reports. This feature allows you to share data during assemblies or planning sessions.
Is my data secure?
Yes. The system uses secure authentication and stores all data in a protected database. Only authorized SK officials with login credentials can access the system.
What should I do if I forgot my password?
Contact your System Administrator or SK Chairperson to reset your password. They have the authority to manage user accounts and credentials.
Does the mobile app work offline?
Yes. The mobile app (when available) can work offline during data collection. Once the device connects to the internet, all collected data will automatically sync to the web system.
How often should we update youth profiles?
It is recommended to conduct profiling at least once a year or whenever there are significant changes in youth status (e.g., graduation, employment, change of address).
